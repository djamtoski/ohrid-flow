# Ohrid Loyalty Flow

A mobile-only digital loyalty stamp card application for small businesses in Ohrid, Macedonia.

## Project Structure

- `mobile-app/` - Ionic Angular mobile application (Android & iOS)

## Quick Start

See [PROJECT_SUMMARY.md](./PROJECT_SUMMARY.md) for complete project overview and setup instructions.

## Mobile App

Navigate to `mobile-app/` directory and follow the setup instructions in `mobile-app/README.md`.

```bash
cd mobile-app
npm install
npm start
```

## Features

- **Owner Mode**: Manage business, scan QR codes, view customers, customize theme
- **Customer Mode**: View loyalty card, display QR code, track rewards
- **Dynamic Theming**: Customizable colors, logos, and branding
- **QR Code System**: Scan and generate QR codes for stamp collection
